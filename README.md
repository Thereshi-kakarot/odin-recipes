# odin-recipes
This project is a quick recipe web project.
In this project the main goal is that users should be able to see the main picture of their desired meal, unordered list of their ingredients, and lastly, ordered list of steps to take when preparing their recipe.

To complete this project, I have managed to be so familiar with working with images, using img tag, links using li tags, ordered lists using ol tags and unordered lists using ul tag.
